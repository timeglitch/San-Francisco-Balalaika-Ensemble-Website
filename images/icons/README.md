# Icons Directory

Place your icon files in this directory.

## Suggested Icons

- `favicon.ico` - Browser favicon (16x16, 32x32)
- `apple-touch-icon.png` - iOS home screen icon (180x180)
- `social-icon.png` - Social media sharing image (1200x630)
- `balalaika-icon.svg` - Custom balalaika instrument icon
- `music-note.svg` - Music-related icons

## Icon Resources

- [Font Awesome](https://fontawesome.com/) - Free icon library
- [Heroicons](https://heroicons.com/) - Free SVG icons
- [Flaticon](https://www.flaticon.com/) - Custom icon search
