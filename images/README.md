# Images Directory

Place your photos and images in this directory.

## Suggested Organization

- `logo.png` - Ensemble logo for the header
- `hero-bg.jpg` - Background image for the hero section
- `about-photo.jpg` - Group photo for the About section
- `performance-*.jpg` - Performance photos
- `gallery/` - Additional gallery images

## Recommended Image Sizes

- Logo: 200-300px wide
- Hero background: 1920x1080px or larger
- Section photos: 800-1200px wide
- Thumbnails: 400x300px

## Supported Formats

- JPEG (.jpg, .jpeg) - For photos
- PNG (.png) - For logos and graphics with transparency
- WebP (.webp) - For optimized web images
- SVG (.svg) - For scalable icons
